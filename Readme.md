GOST'S EYE For System Hardware Information

# Requirements

```
=> nodejs

```

# How to install

```
git clone https://github.com/kpndevroot/gosteye.git
cd gosteye
npm install
```

### How to run it in windows

(make sure you are administrator)

```
npm start
or
node gosteye.js

```

### How to run it in linux / mac

```
sudo npm start
or
sudo node gosteye.js

```
